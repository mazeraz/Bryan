Can you help me in simple way change div 1 and show div 2 back in forth on button click with HTML CSS and JAVASCRIPT
ChatGPT said:

Sure, I can help you with that! Below is a simple example that toggles between two <div> elements when a button is clicked using HTML, CSS, and JavaScript.

HTML

html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Toggle Divs</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <button id="toggleButton">Toggle Divs</button>
    <div id="div1" class="box">This is Div 1</div>
    <div id="div2" class="box hidden">This is Div 2</div>

    <script src="script.js"></script>
</body>
</html>

CSS (styles.css)

css
.box {
    width: 200px;
    height: 100px;
    background-color: lightblue;
    margin-top: 10px;
    display: flex;
    align-items: center;
    justify-content: center;
    border: 1px solid #333;
}

.hidden {
    display: none;
}

JavaScript (script.js)

javascript
document.getElementById('toggleButton').addEventListener('click', function() {
    const div1 = document.getElementById('div1');
    const div2 = document.getElementById('div2');

    // Toggle visibility by adding or removing the 'hidden' class
    div1.classList.toggle('hidden');
    div2.classList.toggle('hidden');
});

How It Works:

  1. HTML: There are two <div> elements and a button. Only one <div> is visible at a time.
  2. CSS: The .hidden class hides the element by setting display: none;.
  3. JavaScript: An event listener is added to the button to toggle the .hidden class on each <div> when the button is clicked.

Clicking the button will switch between showing Div 1 and Div 2!

JavaScript (script.js)

javascript
const showDiv1Button = document.getElementById('showDiv1Button');
const showDiv2Button = document.getElementById('showDiv2Button');
const div1 = document.getElementById('div1');
const div2 = document.getElementById('div2');

showDiv1Button.addEventListener('click', function() {
    div1.classList.remove('hidden');
    div1.classList.add('show');
    div2.classList.remove('show');
    div2.classList.add('hidden');

    // Add selected class to showDiv1Button and remove it from showDiv2Button
    showDiv1Button.classList.add('selected');
    showDiv2Button.classList.remove('selected');

showDiv2Button.addEventListener('click', function() {
    div2.classList.remove('hidden');
    div2.classList.add('show');
    div1.classList.remove('show');
    div1.classList.add('hidden');

    // Add selected class to showDiv2Button and remove it from showDiv1Button
    showDiv2Button.classList.add('selected');
    showDiv1Button.classList.remove('selected');
});

Explanation:

  1. CSS:
    • .box elements start with opacity: 0 to ensure they are initially hidden.
    • The .fade-in.show class sets opacity: 1 and restores display to flex, which applies the fade-in effect with a transition.
    • The transition property (transition: opacity 0.5s ease-in-out;) creates the fade-in effect.
  2. JavaScript:
    • When toggling visibility, the .show class is added, which triggers the fade-in effect, and the .hidden class is removed.
    • Conversely, when hiding an element, the .hidden class is added, and the .show class is removed.

With these changes, the items inside the <div> will fade in smoothly when toggled!
